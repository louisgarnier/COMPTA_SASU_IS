## Lancement & tests
| Commande | Effet |
|---|---|
| `make dev` | uvicorn `:8000` (reload) + `next dev` `:3000` en parallèle (Ctrl-C arrête les deux). |
| `make test` | `pytest` (back) + `jest` (front). |
| `make back` / `make front` | un seul des deux services. |
| `make install` | deps back (pip dans venv) + front (npm). |
